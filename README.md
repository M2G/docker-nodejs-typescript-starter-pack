# Docker, NodeJS and Typescript Starter

## Features
* Docker
* NodeJS (+14)
* Typescript

## Getting Started

Run in development:

```sh
$ npm run watch-debug
```

with docker:

```sh
$ docker-compose up -d
```

Run in production:
```sh
$ run.sh
```
or
```sh
$ npm run build
$ npm run serve
```

Run linter:
```sh
$ npm run tslint
```

Run tests:
```sh
$ npm run test
```
